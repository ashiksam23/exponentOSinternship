# ExponentOS — XP & Leveling System

## How XP Works

Every intern starts at 0 XP on Day 1. XP is earned through building, competing, collaborating, and showing up. XP is tracked on a shared spreadsheet visible to all interns.

## Earning XP

### Daily XP

| Action | XP | Notes |
|--------|-----|-------|
| Show up on time | +5 | Must be present at Kickoff |
| Complete the Build task | +15 | Mentor confirms completion |
| Share during Share & Wrap | +5 | Even "I struggled with X" counts |
| Help a peer | +10 | Mentor or peer confirms |
| Ask a great question | +5 | Mentor discretion |

### Play Block XP

| Placement | XP |
|-----------|-----|
| 1st place | +25 |
| 2nd place | +20 |
| 3rd place | +15 |
| Participated | +10 |

### Milestone Bonus XP

| Action | XP | Notes |
|--------|-----|-------|
| Day 4 mini-showcase submitted | +20 | Prompt, creative AI, and reflection artifacts |
| Day 7 builder sprint completed | +20 | Chatbot or no-code tool is shareable |
| Day 10 "Hello AI" script works | +25 | Python successfully calls the Gemini API |
| Capstone plan approved | +20 | Day 11 scope is realistic and demoable |
| Capstone demo-ready | +30 | Core flow works by Day 13 |
| Final Demo presentation | +25 | Everyone who presents |
| Final Demo audience choice | +15 | Peer-voted bonus |
| 5-session attendance streak | +20 | Award every 5 consecutive attended sessions |

### Special XP (Anytime)

| Action | XP | Notes |
|--------|-----|-------|
| First Bug Fix | +10 | One-time, first time debugging |
| Taught someone something | +15 | Peer confirms they learned |
| Went above and beyond | +20 | Mentor discretion — something unexpected |
| Found a creative solution | +15 | Did something no one else thought of |

## Titles (Leveling Up)

| Title | XP Required | Unlocked |
|-------|------------|----------|
| **Apprentice** | 0 | Everyone starts here |
| **Builder** | 150 | Usually around Day 4 |
| **Hacker** | 350 | Usually around Day 9 |
| **AI Creator** | 600 | Usually around Final Demo |
| **Legend** | 800+ | For those who truly go above and beyond |

## Leaderboard Categories

The leaderboard isn't just "most XP." Multiple categories ensure different strengths are recognized:

| Category | What It Tracks |
|----------|---------------|
| **Overall XP** | Total points |
| **Build Champion** | Most Build task XP |
| **Play Champion** | Most Play block wins |
| **Community Star** | Most "helped a peer" and "taught someone" XP |
| **Streak Master** | Longest perfect attendance streak |
| **Ship Mode** | Most capstone and portfolio milestone XP |

## Rules

- XP can never be taken away (no negative XP)
- Ties are celebrated, not broken
- Mentors award XP transparently — explain why
- Interns can nominate peers for "helped me" XP
- XP is for motivation, not ranking people's worth — keep it fun
