# ExponentOS — Badges

Badges are awarded for specific achievements. Once earned, they're permanent. Interns collect them on their profile throughout the program.

## Program Badges

### AI Foundations Badges
| Badge | How to Earn |
|-------|-------------|
| **Prompt Whisperer** | Best prompt engineer during Days 1-2 |
| **AI Artist** | Most creative AI-generated visual from Day 3 |
| **Storyteller** | Strongest AI-assisted story or post from Day 3 |
| **Myth Buster** | Best contribution during the Day 4 AI ethics discussion |
| **Explorer** | Tried the most different AI tools in the opening sprint |

### AI Builder Badges
| Badge | How to Earn |
|-------|-------------|
| **Bot Boss** | Best chatbot prototype |
| **Tool Smith** | Most useful no-code AI tool |
| **Bug Hunter** | Found the most helpful failure case during stress testing |
| **Speed Demon** | Won the no-code Speed Build challenge |
| **Pitch Perfect** | Clearest short project pitch |

### Code + API Badges
| Badge | How to Earn |
|-------|-------------|
| **Code Newbie** | Everyone who writes their first working Python program |
| **Clean Coder** | Cleanest, most readable code (mentor picked) |
| **Puzzle Master** | Won the logic or debugging challenge |
| **API Explorer** | Found and used the most interesting free API |
| **Hello AI** | First successful Python-to-Gemini script |

### Capstone + Portfolio Badges
| Badge | How to Earn |
|-------|-------------|
| **AI Architect** | Best capstone project (peer voted) |
| **Portfolio Pro** | Best-looking portfolio site |
| **Code Reviewer** | Gave the most helpful peer feedback |
| **Ship It** | First to deploy their portfolio live |
| **Valedictorian** | Highest overall XP at graduation |

## Special Badges (Earned Anytime)

| Badge | How to Earn |
|-------|-------------|
| **First Blood** | Fixed your first bug |
| **Helping Hand** | Helped another intern solve a problem (3 times) |
| **Peer Mentor** | Consistently helped others throughout the program |
| **Curiosity Champion** | Asked the most useful questions across the program |
| **Comeback Kid** | Struggled with something, then crushed it later |
| **After Hours Builder** | Worked on a project outside of session hours |
| **Trendsetter** | Built something no one else thought of |
| **Perfect Streak** | Perfect attendance for 5+ consecutive sessions |
| **The Finisher** | Completed every single Build task in the program |

## How Badges Are Displayed

- Mentors announce badges during mini-showcases and Final Demo
- Badges are listed on each intern's portfolio page
- Interns can choose their "top 3" badges to feature
- A badge wall is maintained in the shared chat/Discord

## Badge Design

Each badge should have:

- A short name (above)
- An emoji icon (for chat/Discord display)
- A one-line description (for the portfolio)
