# ExponentOS — AI Internship Program

> **Build. Play. Learn.** A 14-day AI internship for ages 15-18.

## What is ExponentOS?

ExponentOS is a free, fully remote internship program that takes teenagers from zero coding experience to building and publishing an AI-powered capstone project. From **Tuesday, May 12, 2026** through **Friday, May 29, 2026**, interns go from "What is AI?" to launching a public portfolio that showcases their capstone and the practice projects that helped them get there.

## Program at a Glance

| Detail | Info |
|--------|------|
| **Duration** | 14 weekday sessions |
| **Dates** | May 12-29, 2026 |
| **Daily Time** | 3 hours/day |
| **Format** | Fully remote |
| **Ages** | 15-18 |
| **Prerequisites** | None — absolute beginners welcome |
| **Cost** | Free (all tools are free-tier) |
| **Outcome** | Public GitHub Pages portfolio centered on one AI-powered capstone |

## The 14-Day Journey

| Days | Dates | Theme | What You Build |
|------|-------|-------|----------------|
| **Days 1-4** | May 12-15 | AI Foundations | Prompt recipe book, creative AI mini-showcase, responsible AI reflection |
| **Days 5-7** | May 18-20 | AI Builder Sprint | Chatbot prototype + small no-code AI tool |
| **Days 8-10** | May 21-25 | Code + AI APIs | Python mini-project, API practice script, first Gemini-powered Python script |
| **Days 11-14** | May 26-29 | Capstone + Portfolio | AI-powered capstone app, published portfolio, final presentation |

## Daily Session Flow

Every session follows the same rhythm so interns know what to expect:

```
0:00 - 0:10  Kickoff        Quick recap, goals, warm-up
0:10 - 0:50  Learn          Concept intro + live demos
0:50 - 1:00  Break
1:00 - 1:50  Build          Hands-on project work
1:50 - 2:00  Break
2:00 - 2:40  Play           Challenges, competitions, peer fun
2:40 - 3:00  Share & Wrap   Show work, shout-outs, next-session preview
```

## Gamification

Interns earn **XP**, unlock **badges**, climb the **leaderboard**, and level up through titles:

**Apprentice -> Builder -> Hacker -> AI Creator**

Points come from building, winning challenges, helping peers, and showing up on time. Multiple leaderboard categories ensure different strengths shine.
