# ExponentOS — Mentor Daily Checklist

Use this checklist before, during, and after every session.

## Before Session (15 min prior)

- [ ] Open video call link, test audio/video
- [ ] Open all tools you'll demo (ChatGPT, Replit, etc.)
- [ ] Review today's session plan
- [ ] Open XP tracking spreadsheet
- [ ] Have today's warm-up question ready
- [ ] Check if any interns messaged with questions/issues

## During Session

### Kickoff
- [ ] Started on time
- [ ] Asked warm-up question
- [ ] Gave 2-sentence recap of yesterday
- [ ] Stated today's goal clearly

### Learn Block
- [ ] Kept lecture under 30% of the time
- [ ] Did live demo (not slides)
- [ ] Paused for intern participation at least twice
- [ ] Checked in: "Does this make sense?"

### Build Block
- [ ] Guided portion first, then independent
- [ ] Checked on every intern at least once
- [ ] No one was stuck for more than 5 minutes
- [ ] Gave stretch challenges to fast finishers

### Play Block
- [ ] Explained rules clearly before starting
- [ ] Kept time strictly
- [ ] Awarded XP immediately after
- [ ] Celebrated effort, not just winners

### Share & Wrap
- [ ] Everyone shared something
- [ ] Gave specific shout-outs (by name)
- [ ] Previewed tomorrow
- [ ] Ended on time (not over)

## After Session (15 min after)

- [ ] Updated XP spreadsheet
- [ ] Noted struggling interns (reach out privately)
- [ ] Noted excelling interns (stretch challenges tomorrow)
- [ ] Sent follow-up materials if needed
- [ ] Reviewed tomorrow's session plan briefly
