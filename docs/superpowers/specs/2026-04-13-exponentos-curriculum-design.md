# ExponentOS AI Internship — Curriculum Design Spec

**Date:** 2026-04-13  
**Updated:** 2026-05-11  
**Status:** Approved

## Overview

A 14-session, fully remote AI internship program for ages 15-18 with zero prior coding experience. The program runs from **Tuesday, May 12, 2026** through **Friday, May 29, 2026** and uses a "Dual Track" approach: every session splits between a Build Track (project-focused, no-code to code progression) and a Play Track (gamified challenges, creative exploration, social elements).

## Key Decisions

- **Duration:** 14 weekday sessions, 3 hours/day
- **Dates:** May 12-29, 2026
- **Format:** Fully remote
- **Audience:** Ages 15-18, absolute beginners
- **Approach:** Dual Track (Build + Play every session)
- **Tools:** Free-tier only
- **Primary deliverable:** Public GitHub Pages portfolio centered on one AI-powered capstone
- **Supporting artifacts:** Prompt recipe book, creative AI sample, chatbot or no-code tool, and Python/API practice project
- **Theme:** Build + Fun + Learning
- **Branding:** ExponentOS

## Daily Session Structure

| Time | Block | Purpose |
|------|-------|---------|
| 0:00-0:10 | Kickoff | Recap, goals, warm-up |
| 0:10-0:50 | Learn | Concept intro, live demos (30% lecture, 70% follow-along) |
| 0:50-1:00 | Break | |

| 1:50-2:00 | Break | |

| 2:40-3:00 | Share & Wrap | Show work, shout-outs, preview |

## Gamification System

| Element | Description |
|---------|-------------|
| XP Points | Earn for builds, challenges, helping peers, attendance |
| Badges | Awarded at milestones and Final Demo |
| Leaderboard | Daily updates, multiple categories |
| Titles | Apprentice -> Builder -> Hacker -> AI Creator |
| Team Challenges | Randomized teams for Play block, rotating |

## Tools (All Free Tier)

- ChatGPT, Google Gemini, Microsoft Copilot
- Ideogram (AI art)
- Coze/Poe (chatbot builders)
- Google Sheets, Canva AI, Google Docs
- Replit (Python IDE)
- Google Gemini API (free tier)
- GitHub + GitHub Pages

## Portfolio Deliverables

1. **AI-powered capstone:** A Python app that uses the Gemini API and solves a clear user problem.
2. **Portfolio website:** A GitHub Pages site showcasing the capstone, supporting artifacts, badges, and reflection.
3. **Supporting artifacts:** Prompt recipe book, creative AI sample, chatbot or no-code tool, and Python/API practice project.

## Compression Strategy

- Preserve the full learning arc from AI user to AI builder to AI creator.
- Combine writing and visual AI into one creative session.
- Combine chatbot design/build/upgrade into two sessions.
- Combine Python fundamentals into two focused sessions.
- Protect the final five sessions for Gemini API setup, capstone planning, build, polish, portfolio, and final demo.
