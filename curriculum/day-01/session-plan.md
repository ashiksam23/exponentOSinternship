# Day 1 — Welcome to AI

**Date:** Tuesday, May 12, 2026  
**Goal:** By the end of today, interns will have set up core tools, tried multiple AI assistants, and captured first impressions about what AI can and cannot do.

## Kickoff (0:00 - 0:10)

**Warm-up:** "What is one thing you hope AI can help you build by May 29?"

**Today's goal:** "By the end of today, you'll have talked to multiple AIs, made them do something surprising, and set up the tools you'll use for the next 14 sessions."

## Learn Block (0:10 - 0:50)

### Concepts
- What generative AI is: pattern prediction, not magic
- Common AI tools: ChatGPT, Gemini, Copilot
- What AI is good at: drafting, brainstorming, summarizing, tutoring, coding help
- What AI is risky at: facts, bias, privacy, overconfidence
- How the program works: Build + Play every day, capstone-first portfolio

### Live Demo
Ask the same prompt in ChatGPT, Gemini, and Copilot:

> "Explain AI to a 15-year-old using an example from school."

Compare tone, usefulness, accuracy, and follow-up behavior.

### Pause Points
- "Which answer was clearest?"
- "Which answer sounded most confident?"
- "What would you ask next to make it better?"

## Build Block (1:00 - 1:50)

### Project: Tool Setup + AI Exploration Notes

1. Confirm access to ChatGPT, Gemini, Copilot, Replit, GitHub, and Ideogram.
2. Create a simple notes document titled `ExponentOS AI Exploration`.
3. Test 3 prompts:
   - Explain a hard school topic
   - Plan a weekend activity
   - Generate a project idea for this internship
4. For each prompt, record:
   - Tool used
   - Best response
   - What was helpful
   - What was questionable or missing

**Stretch:** Ask the same prompt in two tools and write which one you would trust more and why.

## Play Block (2:00 - 2:40)

### Activity: AI Scavenger Hunt

Find an AI response that:

- Teaches something clearly
- Makes a creative suggestion
- Gives a wrong or suspicious answer
- Uses a useful analogy
- Changes after you give better instructions

Award XP for each find and a bonus for the best "AI got this wrong" catch.

## Share & Wrap (2:40 - 3:00)

**Share prompt:** "Show us one AI response from today that surprised you."

**Next-session preview:** "Tomorrow we learn how to control AI better with prompt engineering."

## Post-Session Notes

- Confirm every intern can access the required tools.
- Log any account issues for follow-up before Day 2.
- Award Explorer-style XP for scavenger hunt participation.
