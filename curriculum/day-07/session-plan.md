# Day 7 — No-Code AI Tool Sprint

**Date:** Wednesday, May 20, 2026  
**Goal:** By the end of today, interns will have built a small AI-powered tool or automation without writing code.

## Kickoff (0:00 - 0:10)

**Warm-up:** "What repetitive task would you love to automate?"

**Recap:** Yesterday we built chatbots. Today we use AI inside familiar tools to solve a smaller workflow problem.

**Today's goal:** "Build a useful mini tool, then explain who it helps and why."

## Learn Block (0:10 - 0:50)

### Concepts
- AI tool vs chatbot: forms, sheets, docs, templates, workflows
- Inputs and outputs
- Good tools reduce repeated work
- Simple UX: clear labels, examples, and next steps
- Scope control: one user, one job, one output

### Live Demo
Build a "Study Plan Generator" in a doc or sheet:

- Inputs: subject, deadline, available time
- AI prompt template
- Output: 5-day study plan
- Add a quality checklist

### Pause Points
- "What input is required?"
- "What output should be predictable?"
- "How will a user know what to do next?"

## Build Block (1:00 - 1:50)

### Project: Mini AI Tool

Pick one:

- Study planner
- Quiz generator
- Meal planner
- Resume bullet improver
- Birthday message generator
- Travel idea recommender
- Custom tool for your own problem

Your tool must include:

- User instructions
- Required inputs
- AI prompt or formula
- Example output
- One quality check

**Stretch:** Add a simple visual design in Canva or Google Docs.

## Play Block (2:00 - 2:40)

### Activity: Speed Build

30-minute challenge:

1. Choose a tiny problem.
2. Build the simplest useful version.
3. Trade with a peer for testing.
4. Make one improvement.

## Share & Wrap (2:40 - 3:00)

**Share prompt:** "Who is your tool for, and what task does it make easier?"

**Next-session preview:** "Tomorrow we start writing Python code from scratch."

## Post-Session Notes

- Award builder sprint completion XP.
- Encourage interns to keep the chatbot or no-code tool as a supporting portfolio artifact.
