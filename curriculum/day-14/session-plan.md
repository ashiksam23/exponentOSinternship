# Day 14 — Final Demo + Graduation

**Date:** Friday, May 29, 2026  
**Goal:** By the end of today, interns will present their capstone, share their portfolio, receive awards, and leave with clear next steps.

## Kickoff (0:00 - 0:10)

**Warm-up:** "In one word, how do you feel about presenting what you built?"

**Recap:** Over 14 sessions, you explored AI, built with no-code tools, learned Python, connected to Gemini, and created a capstone.

**Today's agenda:** "Final polish, final presentations, awards, and what to learn next."

## Learn Block (0:10 - 0:50)

### Topic: Career Paths in AI & What to Learn Next

Share 5 possible paths:

- AI product builder
- Software developer
- Data analyst
- UX/product designer
- AI safety/responsible AI researcher

### Next Skills
- Python practice
- Web basics: HTML, CSS, JavaScript
- APIs and data
- Prompt design for real workflows
- GitHub and project documentation
- Responsible AI habits

### Pause Points
- "Which path sounds most interesting?"
- "What skill do you want to practice next?"
- "What would you build if you had one more month?"

## Build Block (1:00 - 1:50)

### Final Polish + Presentation Prep

Final checklist:

- Capstone runs at least once
- API key is not visible
- Portfolio link or local draft is ready
- Demo script is 2 minutes or less
- One backup screenshot is saved
- Reflection is written

Presentation structure:

1. Problem: Who is this for?
2. Build: What did you make?
3. AI: How does AI help?
4. Demo: Show it working.
5. Reflection: What did you learn?

## Play Block (2:00 - 2:40)

### Graduation Demo Day & Awards

Each intern gets:

- 2 minutes to present
- 1 minute for applause/questions
- Peer votes for audience choice awards

Awards can include:

- Best Capstone
- Most Useful App
- Best Demo
- Best Portfolio
- Most Helpful Peer
- Highest XP
- Biggest Comeback

## Share & Wrap (2:40 - 3:00)

### Final Send-Off

Closing message:

> "Fourteen sessions ago, you were starting from curiosity. Now you have used AI tools, written Python, called an AI API, and shipped a project you can show people. The tools will keep changing. The builder mindset is yours to keep."

**Final prompt:** "Drop your portfolio or project link, plus one thing you want to build next."
