# Day 2 — Prompt Engineering

**Date:** Wednesday, May 13, 2026  
**Goal:** By the end of today, interns will know how to craft effective prompts and will have a tested prompt recipe book.

## Kickoff (0:00 - 0:10)

**Warm-up:** "What was the most useful or weirdest AI response you got after Day 1?"

**Recap:** Yesterday we explored what AI can do. Today we learn how to make AI more useful on purpose.

**Today's goal:** "You're going to build a prompt recipe book you can reuse for the rest of the program."

## Learn Block (0:10 - 0:50)

### Concepts
- Prompt = instructions you give the AI
- Five prompt power-ups:
  1. Be specific
  2. Give context
  3. Assign a role
  4. Give examples
  5. Ask it to revise
- Constraints: tone, length, format, audience
- Iteration: first answer is a draft, not the finish line

### Live Demo
Start with:

> "Help me study."

Improve it step by step until it becomes:

> "Act as a friendly biology tutor. Create a 20-minute study plan for a 10th grader reviewing photosynthesis. Include 5 practice questions and explain wrong answers simply."

### Pause Points
- "Which added detail changed the answer most?"
- "When should you ask AI for a table?"
- "What should you never paste into a prompt?"

## Build Block (1:00 - 1:50)

### Project: Prompt Recipe Book

Create 8 tested prompts:

- 2 study prompts
- 2 creative prompts
- 2 productivity prompts
- 1 coding-helper prompt
- 1 capstone-idea prompt

For each recipe, include:

- Purpose
- Prompt
- Best result
- One improvement you made after testing

**Stretch:** Add a "bad prompt vs better prompt" comparison.

## Play Block (2:00 - 2:40)

### Activity: Prompt Battle

Teams compete to get the best AI answer for a shared challenge:

> "Design a beginner-friendly app idea that helps students manage stress."

Rounds:

1. Best clarity
2. Most creative
3. Most realistic to build in 14 days
