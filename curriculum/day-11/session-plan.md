# Day 11 — Capstone Planning

**Date:** Tuesday, May 26, 2026  
**Goal:** By the end of today, interns will choose a capstone idea, define a realistic feature scope, and start the project skeleton.

## Kickoff (0:00 - 0:10)

**Warm-up:** "What problem do you want your final AI app to solve?"

**Recap:** Yesterday Python successfully talked to Gemini. Today we turn that power into a focused capstone.

**Today's goal:** "Choose a capstone idea that is useful, personal, and small enough to demo on Friday."

## Learn Block (0:10 - 0:50)

### Concepts
- Capstone formula: user + problem + AI help + clear output
- Good scope: one input flow, one AI call, one useful result
- MVP vs extra features
- Demo-first thinking: what will the audience see in 2 minutes?
- Project brief: name, user, problem, features, risks

### Live Demo
Turn a vague idea into a scoped project:

Vague: "AI homework app"

Scoped: "A Python study coach that asks for a topic and test date, then generates a 5-day study plan and 5 practice questions."

### Pause Points
- "What can be finished by Day 13?"
- "What feature should be cut first if time is short?"
- "What makes the demo easy to understand?"

## Build Block (1:00 - 1:50)

### Project: Capstone Brief + Skeleton

Create a project brief:

- Project name
- Target user
- Problem
- Input(s)
- AI prompt role
- Output
- Must-have features
- Nice-to-have features
- Demo script in 3 steps

Then start a Replit project with:

- `main.py`
- placeholder user input
- placeholder AI prompt
- comments for next steps

**Stretch:** Write your README draft with title, problem, tools, and planned demo.

## Play Block (2:00 - 2:40)

### Activity: Idea Pitch Tournament

Each intern gives a 45-second pitch:

- Who it helps
- What it does
- Why AI is useful
- What will be demoed Friday

Peers vote for most useful, most creative, and most realistic.

## Share & Wrap (2:40 - 3:00)

**Share prompt:** "State your capstone in one sentence: My app helps ___ do ___ by using AI to ___."

**Next-session preview:** "Tomorrow is the main capstone build day. Come ready to finish the core flow."

## Post-Session Notes

- Approve or tighten every capstone scope before the session ends.
- Award capstone plan milestone XP.
