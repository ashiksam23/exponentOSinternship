# Day 10 — Hello AI with Gemini

**Date:** Monday, May 25, 2026  
**Goal:** By the end of today, interns will have set up the Gemini API and written a Python script that sends a prompt to an AI model.

## Kickoff (0:00 - 0:10)

**Warm-up:** "What should happen when Python and AI work together?"

**Recap:** Last session our Python code pulled data from the internet. Today it sends a prompt to an AI model and gets a response back.

**Today's goal:** "Your Python program will talk to AI without using a browser."

## Learn Block (0:10 - 0:50)

### Concepts
- AI API = a way for code to send prompts and receive model responses
- API keys are private passwords for services
- Environment variables/Replit Secrets keep keys out of code
- Request -> response -> display result
- Prompt design still matters inside code
- Keep first AI scripts small and testable

### Live Demo
Create a "Hello AI" script:

- Store the Gemini API key in Replit Secrets
- Install or import the required library
- Send a short prompt
- Print the model response
- Change the prompt and compare results

### Pause Points
- "Where should the API key live?"
- "What part of the code is the prompt?"
- "How could this become a useful app?"

## Build Block (1:00 - 1:50)

### Project: First AI-Powered Python Script

Build one:

- AI study coach
- Joke explainer
- Idea generator
- Friendly writing helper
- Simple chatbot loop

Requirements:

- Reads user input
- Sends a prompt to Gemini
- Prints the response
- Does not expose the API key in code or screenshots

**Stretch:** Add a system-style instruction that gives the AI a role and tone.

## Play Block (2:00 - 2:40)

### Activity: AI Roulette

Each intern draws a random role, audience, and format:

- Role: tutor, coach, editor, inventor
- Audience: 6th grader, parent, gamer, teacher
- Format: list, poem, table, pep talk

Use Python to send the combined prompt and compare outputs.
