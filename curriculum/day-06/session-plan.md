# Day 6 — Chatbot Prototype

**Date:** Tuesday, May 19, 2026  
**Goal:** By the end of today, interns will have a working chatbot prototype and at least one improvement from peer testing.

## Kickoff (0:00 - 0:10)

**Warm-up:** "What is one thing your bot must do well today?"

**Recap:** Yesterday we designed bots with users, personalities, and limits. Today we build and test them.

**Today's goal:** "By the end of this session, someone else can talk to your chatbot and give feedback."

## Learn Block (0:10 - 0:50)

### Concepts
- Coze/Poe basics: bot name, description, instructions, sample prompts
- System prompt vs user message
- Knowledge base: facts your bot should rely on
- Testing for edge cases
- Iterating after feedback

### Live Demo
Build a simple "Club Finder Bot":

- Add name and description
- Paste system instructions
- Add 5 facts as knowledge
- Test normal, weird, and unsafe questions
- Revise the instructions

### Pause Points
- "What did the bot misunderstand?"
- "What one sentence would make the instructions clearer?"
- "What should the bot do when it does not know?"

## Build Block (1:00 - 1:50)

### Project: Chatbot Prototype

1. Build your bot in Coze or Poe.
2. Add your system prompt from Day 5.
3. Add a short knowledge base or sample Q&A.
4. Test at least 5 user questions.
5. Revise the bot after testing.
6. Save the share link or screenshots.

**Stretch:** Add a welcome flow that asks the user one clarifying question before giving advice.

## Play Block (2:00 - 2:40)

### Activity: Stress Test Challenge

Peers test each bot with:

- A normal request
- A vague request
- An off-topic request
- A request outside the bot's limits
- A trick question

Award XP for helpful testers and thoughtful fixes.

## Share & Wrap (2:40 - 3:00)

**Share prompt:** "What was the biggest improvement you made after peer testing?"

**Next-session preview:** "Tomorrow we build a small AI tool that is not a chatbot."

## Post-Session Notes

- Collect chatbot links or screenshots.
- Award Bot Boss and Bug Hunter candidates.
