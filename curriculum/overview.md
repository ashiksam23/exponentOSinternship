# ExponentOS 14-Day Curriculum Overview

> From "What is AI?" to a shipped AI-powered capstone in 14 weekday sessions.

## Program Goal

Interns learn practical AI literacy, prompt engineering, no-code AI app design, beginner Python, AI APIs, and portfolio publishing. The main outcome is a demo-ready capstone project that uses the Gemini API, supported by smaller practice artifacts from earlier sessions.

## Capstone-First Portfolio

The portfolio should feature:

- **Primary project:** AI-powered capstone using Python and the Gemini API
- **Supporting artifacts:** prompt recipe book, creative AI sample, chatbot or no-code tool, and Python/API practice project
- **Reflection:** what the intern learned, what was hard, and what they want to build next

## Tools

- ChatGPT, Gemini, Copilot for AI exploration and prompting
- Ideogram for image generation
- Coze or Poe for chatbot prototyping
- Google Sheets, Docs, and Canva for no-code tools and presentation assets
- Replit for Python
- Google AI Studio for the Gemini API
- GitHub and GitHub Pages for portfolio publishing

## Key Concepts

- AI strengths, limits, hallucinations, bias, and responsible use
- Prompt structure: role, task, context, examples, constraints, iteration
- Chatbot anatomy: audience, persona, system prompt, knowledge, edge cases
- Beginner Python: variables, input, conditionals, loops, lists, functions
- API basics: requests, responses, JSON, status codes, API keys
- Capstone scope control, peer review, demos, and portfolio storytelling
