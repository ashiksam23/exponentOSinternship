# ExponentOS — Tools Setup Guide

Everything in this program is free. Here's how to get set up before Day 1.

## Required Accounts (Set Up Before Day 1)

### 1. Google Account
- **Why:** Access to Gemini, Google Sheets, Google Docs, and Google AI Studio
- **Setup:** Go to [accounts.google.com](https://accounts.google.com) and create a free account
- Under 18? Use a parent/guardian's account or create one with their permission

### 2. ChatGPT Account
- **Why:** Primary AI tool for exploration, prompting, and project support
- **Setup:** Go to [chat.openai.com](https://chat.openai.com) and sign up (free tier)
- The free tier is all you need

### 3. Microsoft Account (for Copilot)
- **Why:** Alternative AI assistant
- **Setup:** Go to [copilot.microsoft.com](https://copilot.microsoft.com)
- Works with any Microsoft/Outlook account

### 4. GitHub Account
- **Why:** Portfolio hosting, code sharing, and GitHub Pages
- **Setup:** Go to [github.com](https://github.com) and sign up
- Choose a professional username — this will be your portfolio URL

### 5. Replit Account
- **Why:** Python coding environment (no install needed)
- **Setup:** Go to [replit.com](https://replit.com) and sign up with your Google/GitHub account
- Free tier is sufficient

## Program Tools by Phase

### Days 1-4: AI Foundations
- **Ideogram** (AI art): [ideogram.ai](https://ideogram.ai) — sign up free
- **Google Gemini**: [gemini.google.com](https://gemini.google.com) — uses your Google account
- **ChatGPT / Copilot**: Useful for comparing AI responses

### Days 5-7: AI Builder Sprint
- **Coze** (chatbot builder): [coze.com](https://coze.com) — sign up free
  - Alternative: **Poe** at [poe.com](https://poe.com)
- **Canva** (design): [canva.com](https://canva.com) — sign up free
- **Google Sheets / Docs**: Uses your Google account

### Days 8-14: Code, AI APIs, and Capstone
- **Replit** (already set up above)
- **Google AI Studio** (Gemini API): [aistudio.google.com](https://aistudio.google.com)
  - Sign in with Google account
  - Get a free API key (generous free tier)
- **GitHub Pages**: Enabled through your GitHub account

## Browser Requirements

- Use **Google Chrome** or **Microsoft Edge** (latest version)
- Ensure pop-ups are allowed for the tools above
- A stable internet connection is required for all sessions

## Hardware Requirements

- Any computer (Windows, Mac, Chromebook, or Linux)
- Webcam and microphone for remote sessions
- No special hardware or GPU needed

## Checklist

Before Day 1, confirm you can:

- [ ] Log into ChatGPT and send a message
- [ ] Log into Google Gemini and send a message
- [ ] Log into Microsoft Copilot
- [ ] Log into Replit and create a new Python project
- [ ] Log into GitHub
- [ ] Log into Ideogram and generate an image
- [ ] Open your webcam and microphone in your browser

By Day 10, confirm you can:

- [ ] Log into Google AI Studio
- [ ] Create or access a Gemini API key
- [ ] Store the key in Replit Secrets, not directly in code

## Troubleshooting

| Problem | Solution |
|---------|----------|
| "I can't create an account" | Ask a parent/guardian to help — some platforms require age 16+ |
| "Replit is slow" | Try refreshing or using a different browser |
| "ChatGPT says I've hit a limit" | Switch to Google Gemini or Microsoft Copilot — they're interchangeable for most exercises |
| "My API key is not working" | Check that it is copied exactly and stored in Replit Secrets |
| "I don't have a computer" | Contact the program coordinator — we may be able to help |
